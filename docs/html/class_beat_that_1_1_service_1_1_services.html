Detailed Description

Services provides access to global services by interface, e.g. Services.Require<MyServiceInterface>().

Services can be registered explicitly but often the most convenient way is via attribute, e.g.

public interface MyApi {}

/// attribute tells service system to create an instance of MyApiImpl and register it as MyApi [RegisterService(typeof(MyApi)] public class MyApiImpl : MonoBehaviour, MyApi {}

///somewhere Services.Init(() => { Services.Require<MyApi>(); // returns singleton MyApiImpl });
