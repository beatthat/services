enum BeatThat.Service.InterfaceRegistrationPolicy
strong

Policy on which interfaces to register as proxies for the service. Any proxy interface can be used to location or inject the service.

Examples: public interface HasFoo { void Foo(); }

public interface HasBar { void Bar(); }

public class ParentService : HasFoo { public void Foo(); }

public class MyService : HasBar { public void Bar(); }
