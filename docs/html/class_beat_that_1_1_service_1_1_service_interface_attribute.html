Detailed Description

Add this to a service that may be set up as a scene object to make it discoverable/registerable to its (parent) Services, e.g.

[ServiceInterface(typeof(ISomeService))] // this tells the Services component to register this object for interface ISomeService public class SceneObjectService : ISomeService {} // this will live in a game object with config and maybe children
